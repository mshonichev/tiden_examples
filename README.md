# tiden_examples

Playground for Tiden Ignite examples before publication.

== Install pre-requisites

```
    git clone git@github.com:mshonichev/tiden_pkg.git
    cd tiden_pkg
    bash build.sh
```

== Run examples

```
    git clone git@github.com:mshonichev/tiden_examples.git
    cd tiden_examples
    bash run.sh
```
